# RandomTowerDefence
### 3-Lane Roguelike Tower Defense

**세 개의 레인(길)을 동시에 방어하는 타워 디펜스**입니다.  
웨이브를 막아낼 때마다 보상을 선택하며, 선택이 누적되어 **이번 판만의 빌드**가 완성되는 게임을 목표로 합니다.

---

## 어떤 게임인가요?

적들은 3개의 레인을 따라 기지로 향합니다.  
플레이어는 제한된 자원으로 타워를 배치하고, 약한 레인을 보강하며, 점점 강해지는 웨이브를 버텨야 합니다.

웨이브가 끝날 때마다 **3개의 보상 중 1개를 선택**합니다.  
이 선택이 다음 웨이브의 전략을 바꾸고, 매 판 다른 흐름을 만들어냅니다.

---

## 핵심 컨셉

- **3레인 운영:** 한 곳만 잘 막아서는 이길 수 없습니다.
- **선택의 누적:** 매 웨이브의 선택이 빌드를 만들고, 플레이 스타일이 달라집니다.
- **명확한 목표:** 마지막 보스 웨이브를 막아내면 승리합니다.

---

## 승리 / 패배

- **승리:** 마지막 보스 웨이브 방어
- **패배:** 기지 체력이 0이 되면 게임 오버

---

## 게임에서 기대하는 재미

- “지금 이 레인을 살릴까, 다음을 위해 투자할까?” 같은 순간 판단
- 선택지로 인해 매 판 달라지는 성장 경로
- 방어 성공 → 보상 선택 → 더 강한 웨이브 도전의 반복
